# Engineering College Online MCQ Test Portal

This Streamlit application provides an admin and student portal for online MCQ tests with Postgres / NeonDB persistence.

## Features
- Admin login and configuration
- Allowed exam network prefixes for IP-restricted exam access
- Bulk student account upload and password generation
- Branch / semester / class-based question paper uploads
- Time table and exam duration configuration
- Student login by PRN with exam password
- Exam timer, auto-submit warning, and proctoring camera overlay
- Class-wise and branch-wise result download

## Setup
1. Install dependencies:
   ```powershell
   python -m pip install -r requirements.txt
   ```
2. Create a `.env` file from `.env.example` and set `DATABASE_URL`.
3. Start PostgreSQL or NeonDB and ensure the database is reachable.
4. Run:
   ```powershell
   streamlit run app.py
   ```

## Notes
- Admin default credentials are `admin` / `admin123` when no admin exists.
- Student passwords must be generated by admin on exam day.
- Question uploads must include columns: `question`, `a`, `b`, `c`, `d`, `answer`.
- The app stores all data in PostgreSQL / NeonDB.
